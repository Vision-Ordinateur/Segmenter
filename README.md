# Description
Ce projet porte sur la segmentation sémantique basée sur les **Transformers**, inspirée du modèle **Segmenter**.  
L’objectif principal est la **segmentation de la classe personne dans des scènes intérieures**, dans un contexte de **robotique domestique**, où la compréhension précise de l’environnement est essentielle pour la navigation et l’interaction homme-robot.
